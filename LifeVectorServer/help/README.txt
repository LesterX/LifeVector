To build the utilities they each have a shell script that can be run


ENSURE:
****
****
1) You are using BASH as your shell
2) You maintain the folder and file structure
3) You have G++ installed
4) You run the scripts from within the folder (To execute from within the folder: "./scriptnamegoeshere" include the .sh after the scriptname)
5) You execute the scripts from within the top level folder
****
****

To build individually execute any of the following scripts:
buildInterfaceList.sh
buildMemInfo.sh
buildMain.sh
buildOSDets.sh
buildProcessorInfo.sh
buildProcList.sh
buildSysHost.sh


You can also build them all, by executing the following script:
buildAll.sh
